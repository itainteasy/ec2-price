ec2-price
----

Retrieve the latest EC2 price via a command line.

## Installation

```
npm install ec2-price -g
```

## Usage

Output all prices in all regions.

```
$ ec2-price
```

You can specify a region.

```
$ ec2-price --region us-east-1
```
